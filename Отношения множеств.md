# Первая часть
- Представим, что есть множество S студентов какого-нибудь института
- И множество K курсов, которые там проходят
- Теперь найдём прямое произведение S x K и получим все возможные пары (студент-курс)
- Если среди получившегося множества выбрать строки, которые обладают каким-то свойством, например студент s посещает курс k, то получится отношение
- То есть отношение - это подмножество прямого произведения множеств
- Дальше мы будем рассматривать только бинарные отношения - это отношения между двумя множествами
- Также это же самое отношение можно задать с помощью слов, используя подходящие предикаты
- С помощью ориентированного графа, в котором отношение задаётся связями между вершинами
- И с помощью матрицы, где на пересечении стоит либо истина, если связь есть, либо ложь, если её нет
# Вторая часть
- Теорию отношений можно применить к базам данных
- Например, эту таблицу можно представить как четырёхместное отношение, то есть как подмножество прямого произведения четырёх множеств: множества личных номеров, множества фамилий, множества дат и множества адресов
- А эту таблицу можно представить как трёхместное отношение
- Теперь рассмотрим, как одну из операций манипуляции с базой данных можно задать в терминах отношений
- Возьмём операцию соединения. Мы соединяем два предыдущих отношений и получаем новое шестиместное отношение - это подмножество произведения множества, которое было в обеих таблицах, множеств, которые были только в первой таблице, и множеств, которые были только во второй таблице
